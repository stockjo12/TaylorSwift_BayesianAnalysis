# Taylor Swift Song Popularity

This project uses bayesian statistics to try and understand Taylor Swift's popularity on Spotify and YouTube. 

## Usage
1. The "Final Project.qmd" file contains the code run to come to draw conclusions.
3. The "Streaming.csv" file contains the data used in the analysis. 
4. The "Final Project Report.pdf" file contains a written report describing the findings made in the analysis.
